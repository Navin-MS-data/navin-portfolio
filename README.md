# Navin Kumar M.S. — Portfolio

A modern, high-performance personal portfolio website built with **React**, **TypeScript**, and **Vite**. It showcases my skills, experience, and projects as a Front End Developer, featuring a neo-brutalist aesthetic blended with dynamic interactive elements.

---

## 🚀 Features

- **Interactive Hero Section** — Dynamic matrix rain canvas effect and typewriter role animations for a high-tech first impression.
- **Neo-Brutalist Design** — Paper textures, bold borders, drop shadows, and high-contrast monochromatic styling throughout.
- **Interactive Terminal** — A custom-built terminal simulation that lets visitors explore the portfolio in a developer-centric way.
- **GitHub Contributions Graph** — Live visualization of GitHub activity powered by `react-github-calendar`.
- **Direct WhatsApp Contact** — Contact form that pre-fills and opens a WhatsApp conversation for instant communication.
- **Animated Avatar** — Smooth entrance and hover animations on the profile image.
- **Sections** — About, Skills, Projects, Education, Experience, and Contact — all smooth-scrolling with anchor links.
- **Responsive & Accessible** — Fully optimized for mobile, tablet, and desktop viewports with semantic HTML and accessible markup.
- **Lightning-Fast Dev Experience** — Powered by Vite + SWC for instant HMR and optimized production builds.
- **Unit Testing** — Baseline test suite set up with Vitest and React Testing Library.

## 📦 Installation & Setup

To get a local copy up and running, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) ≥ 18 or [Bun](https://bun.sh/)

### Steps

1. **Clone the repository**

   ```bash
   git clone https://github.com/Navin-MS-data/Vara-s--Portfolio.git
   cd Vara-s--Portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or with Bun
   bun install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:5173`.

4. **Build for production**

   ```bash
   npm run build
   ```

5. **Preview the production build**

   ```bash
   npm run preview
   ```
